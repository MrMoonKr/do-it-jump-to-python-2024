## 개발 및 테스트 환경

- windows 10  
- Python 3.12.0
- pip 23.2.1
- venv  
- VS Code  
- ...  

  ```
  $ python -m venv .venv-local
  $ .venv-local/Scripts/Activate.ps1 
  $ (.venv-local) python --version
  $ (.venv-local) pip --version
  $ (.venv-local) pip install -r requirements.txt
  ```
